### Environment
- Python: 3.6.13
- Pytorch: 1.6.0
- Numpy: 1.19.2
- H5py: 2.10.0
- scikit-learn: 0.24.2

### Run Code
```bash
python main.py
```
- if you want to modify configure, you can reference to config.py
- 'emotions.mat' is a data set for demo
